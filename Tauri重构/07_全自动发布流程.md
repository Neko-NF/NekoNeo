# NekoNeo 全自动发布流程

> 版本：v1.0 | 日期：2026-05-10

## 一、发布架构

```
开发者推送 tag → GitHub Actions 触发
    │
    ├── 1. 拉取代码 (checkout)
    ├── 2. 安装 pnpm + Node.js + Rust
    ├── 3. pnpm install → pnpm build (前端)
    ├── 4. tauri build (后端 + 打包)
    │       ├── NSIS 安装包 (.exe)
    │       └── 便携版 (.zip)
    ├── 5. 生成 SHA256SUMS.txt
    ├── 6. 创建 GitHub Release
    │       ├── 上传 .exe + .exe.sig
    │       ├── 上传 .zip  + .zip.sig
    │       ├── 上传 latest.json (更新描述文件)
    │       └── 上传 SHA256SUMS.txt
    └── 7. 客户端自动检测到更新
```

## 二、发布步骤

### 2.1 更新版本号

提交前修改以下文件中的版本号：

| 文件 | 字段 |
|------|------|
| `src-tauri/tauri.conf.json` | `version` |
| `src-tauri/Cargo.toml` | `version` |
| `package.json` | `version` |
| `src/views/About.vue` | 版本字符串 |

### 2.2 打 Tag 并推送

```bash
# Stable 版本
git add -A
git commit -m "release: v1.0.0"
git tag v1.0.0
git push origin main --tags

# Beta 版本
git tag v1.1.0-beta.1
git push origin v1.1.0-beta.1
```

### 2.3 强制更新

在 GitHub Release 的 Release Notes 中包含 `<!-- FORCE_UPDATE -->` 标记，客户端将强制下载并安装，用户无法跳过。

```markdown
## v1.0.1

<!-- FORCE_UPDATE -->

- 修复关键安全漏洞
- 修复数据上报中断问题
```

### 2.4 触发方式

| 方式 | 说明 |
|------|------|
| **Tag 推送** | `v*.*.*` (Stable) / `v*.*.*-beta.*` (Beta) |
| **手动触发** | GitHub Actions → Release → Run workflow → 选择 channel |

## 三、GitHub Secrets 配置

在 `https://github.com/Neko-NF/NekoNeo/settings/secrets/actions` 配置：

| Secret | 说明 |
|--------|------|
| `TAURI_SIGNING_PRIVATE_KEY` | 私钥内容 (`nekoneo.key`) |
| `TAURI_SIGNING_PRIVATE_KEY_PASSWORD` | 私钥密码 |

## 四、客户端更新流程

```
应用启动 (15s 后)
    │
    ├── 检查更新 (GitHub Releases API)
    │     └── 按通道过滤 (stable/beta) → 版本比较
    │
    ├── 有更新
    │     ├── autoDownload = true → 后台静默下载
    │     ├── forceUpdate = true → 强制下载+安装
    │     └── 否则 → Toast 通知用户，用户去设置页手动下载
    │
    ├── 安装
    │     ├── SHA256 校验
    │     ├── 启动 NSIS 安装器
    │     └── 应用退出
    │
    └── 每 30 分钟轮询检查
```

## 五、更新通道

| 通道 | Tag 格式 | 说明 |
|------|---------|------|
| `stable` | `v1.0.0` | 正式发布版 |
| `beta` | `v1.1.0-beta.1` | 测试版 |

## 六、产物清单

每次 Release 生成以下文件：

| 文件 | 用途 |
|------|------|
| `NekoNeo_1.0.0_x64-setup.exe` | NSIS 安装包 |
| `NekoNeo_1.0.0_x64-setup.exe.sig` | 安装包签名 |
| `NekoNeo_1.0.0_x64_en-US.msi` | MSI 安装包 |
| `latest.json` | Tauri Updater 更新描述文件 |
| `SHA256SUMS.txt` | 所有文件的 SHA256 校验 |
